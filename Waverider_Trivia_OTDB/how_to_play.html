How it Works:

Waverider Trivia uses Open Trivia Database (link in about) to generate customizable trivia quizes using questions pulled from the database.

How to Play:

Simply click Take a Quiz from the nav bar. From there click New Game and select the number of questions from the popup menu. Additionally you can select the category and difficulty of the questions. Once all of the questions are answered you will see a modal that will give you your overall score and a prompt to generate a new quiz.

How are my scores saved?

Waverider Trivia stores some data in your browser's memory. This allows high scores to be saved across sessions. For this feature to function properly, please ensure that cookies are enabled on your browser and adblock is disabled for Waverider Trivia.

My quiz didn't populate:

If this occurs, it is because there are not enough quiz questions that meet your specific criteria, thus causing the API to return an error. I apologize for this and a more thoughtful solution would be implemented if this were a commercial platform. However, given the time constraints of this project I unfortunately must just ask that you try a quiz qith different parameters or less questions.
